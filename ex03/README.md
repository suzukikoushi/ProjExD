# 第3回
## 迷路ゲーム　
### ゲーム概要
- ex03/maze.pyを実行すると、1500×900の迷路が表示され、迷路に沿ってこうかとんを移動させるゲーム
- 実行するたびに構造は変わる
### 操作方法
- 矢印キーで操作する
### 追加機能
- 右下の角がゴールになっている
- ゴールに着くと動けなくなる
- 開くたびにこうかとんのイラストが変わる
- issuesでshowinfoでゴール表示が欲しいとあったので実装しました。
- showinfoを閉じると終了します
